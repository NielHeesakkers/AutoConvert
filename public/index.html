    How it works
    When enabled, AutoConvert watches all configured media directories for new .mkv files. When a new file is detected, it waits for the file to stop changing (stability check), then automatically starts conversion. If a conversion is already running, new files are queued.

    Note: When Folder Watch is active, the Scan Interval above is not needed — realtime watching replaces periodic scanning.
